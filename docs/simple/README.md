# 슬쩍 오늘 구현용 문서

이 폴더는 오늘 데모 구현을 위한 간단 문서만 모아둔 곳이다.

## 문서 목록

1. `01_기획서.md`
   - 서비스 컨셉과 데모 범위

2. `02_개발문서.md`
   - 오늘 구현할 개발 구조와 실행 방법

3. `03_프롬프트문서.md`
   - User Profile, Job Profile, Match Result 등 프롬프트

4. `04_AI활용문서.md`
   - GPT/Codex 없이 로컬 파서와 크롤링을 어떻게 쓸지

5. `05_공고데이터_JSON예시.md`
   - 잡코리아 공고 원본, 로컬 매칭 결과, Slack 말투 표시값 저장 예시

## 오늘 기준 핵심 결정

- 실제 운영 서버는 만들지 않는다.
- Vercel 배포 URL을 공유한다. 로컬 데모만 ngrok 링크로 공유한다.
- 로그인은 만들지 않는다.
- 데이터는 `data/state.json`에 저장한다.
- GPT API Key와 로컬 Codex는 사용하지 않는다.
- 추후 개선은 발표 때 설명한다.
